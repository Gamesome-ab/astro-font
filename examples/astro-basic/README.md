# Astro Font Example

Demonstrates `@gamesome/astro-font` with three variable fonts:

- **Rubik Variable** (sans-serif) — applied to `html`
- **Sora Variable** (sans-serif) — applied to `.font-sora`
- **Lora Variable** (serif) — applied to `.font-lora`

Each font has its own page with a comparison showcase: generated font stack, side-by-side fallback comparison, overlap test, width comparison, and metrics table.

## Running

```bash
pnpm install
pnpm dev
```

Then visit `http://localhost:4321` and navigate to each font page.
